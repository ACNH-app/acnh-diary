# Frontend Smoke Test

## 1. 기본 로딩
1. 서버 실행 후 첫 화면이 5초 이내에 표시되는지 확인
2. 상단 네비게이션 버튼(주민/카탈로그 계열)이 보이는지 확인
3. 콘솔 에러가 없는지 확인

## 2. 주민 화면
1. 이름 검색 입력 시 결과가 변경되는지 확인
2. 성격/종 필터 변경 시 결과가 변경되는지 확인
3. 정렬(이름/종/생일/성격) 변경 시 순서가 바뀌는지 확인
4. `좋아하는 주민`, `우리 섬 주민`, `과거 주민` 체크 후 새로고침해도 유지되는지 확인

## 3. 카탈로그 화면
1. 카테고리 전환 시 목록이 바뀌는지 확인
2. `전체/보유/미보유` 탭이 정상 동작하는지 확인
3. 서브카테고리 탭 클릭 시 해당 항목만 필터링되는지 확인
4. `더보기` 버튼 클릭 시 다음 페이지가 append 되는지 확인

## 4. 상세 모달
1. 카드 클릭 시 상세 모달이 열리는지 확인
2. 좌/우 버튼 및 키보드 화살표로 이전/다음 아이템 이동이 되는지 확인
3. ESC 또는 배경 클릭으로 닫히는지 확인
4. 변형 카드 클릭 시 보유 상태가 바뀌는지 확인
5. `전체 보유/전체 해제` 버튼이 정상 동작하는지 확인

## 5. 보조 UI
1. 스크롤 내리면 우하단 `맨 위로` 버튼이 보이는지 확인
2. 버튼 클릭 시 부드럽게 상단으로 이동하는지 확인

## 6. 회귀 확인 포인트
1. 체크/정렬/필터 변경 시 화면이 멈추지 않는지 확인
2. 빠르게 탭 전환해도 데이터가 뒤섞이지 않는지 확인
3. 강력 새로고침(`Cmd/Ctrl + Shift + R`) 후에도 최신 코드가 반영되는지 확인
